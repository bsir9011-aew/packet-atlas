# Packet Atlas v17.0 — Play Mode Memory Drill

v17.0 adds an active recall block to Play Mode.

## Goal

The user should not only read the stage. They should reconstruct it from memory:

- which actor owns the current boundary,
- which layer is active,
- what changed compared with the previous stage,
- what evidence would prove the stage,
- what not to confuse it with.

## UX rule

This block is intentionally small and repeatable. It sits after the animated handoff and before the deeper story/coach sections.
