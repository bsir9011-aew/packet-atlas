# Packet Atlas v9.3 — Guided Read Mode Cleanup

v9.3 removes duplicate reading paths from Focus Mode.

After v9.2, the screen had both:

- the new Guided Step Coach,
- the older summary cards and recommendation line.

That was useful technically, but visually it repeated the same meaning.

## What changed

In Focus Mode:

- the Guided Step Coach becomes the primary reading block,
- the old four summary cards are hidden,
- the old diagnostic trap block is hidden because the coach now carries the reading task,
- the old recommendation note is hidden,
- the old Next choices block is hidden,
- the Coach footer gets a real clickable Next action.

## Product rule

Focus Mode should answer one question:

```text
What do I read and do now?
```

Atlas Mode can still show the full workshop.
