# Packet Atlas v16.0 — Study Snapshot

v16.0 is a larger learning patch for Play Mode.

## What changed

- Adds a visible Study Snapshot panel after the animated motion layer.
- The panel uses the scenario `stage.copy` fields:
  - what the user sees,
  - what really happens,
  - which layer sees it,
  - same payload representation,
  - common confusion,
  - why it matters,
  - analogy.
- Extends PL/EN UI labels for the new panel.
- Extends the Polish scenario dictionary with common network-learning phrases.
- Applies the text display mode more broadly in story, coaching, branch and final recap text.

## Product rule

This still keeps English as the source of truth.

Polish is a learning overlay, not a separate scenario model yet.
