# Play Mode Contract

Play Mode exists for one purpose:

```text
make the current stage impossible to miss
```

A Play screen must answer:

1. What stage am I in?
2. What is happening now?
3. Why does this stage matter?
4. What proof/question confirms it?
5. What should I write down?
6. What is the next stage?

Play Mode must not behave like:

- dashboard,
- inspector,
- workspace catalog,
- scenario catalog,
- metrics screen.

Those belong in Atlas Mode.
