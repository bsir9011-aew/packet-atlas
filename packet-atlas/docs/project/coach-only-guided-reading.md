# Packet Atlas v9.4–v9.5 — Coach-Only Guided Reading

This patch deliberately avoids another dashboard turn.

## v9.4 — Coach-only Focus Mode

Focus Mode keeps the guided coach as the primary reading experience.

It hides dashboard-like supporting sections:

- progress card,
- old summary cards,
- old diagnostic trap and recommendation line,
- failure branch panel,
- progress dot rail.

Atlas Mode can still show the full workshop.

## v9.5 — Final Recap Only

The final recap appears only at the last step.

There are no progress widgets, charts or understanding metrics during the journey.

## Product rule

Focus Mode should feel like this:

```text
read this step
say it simply
answer the proof question
write one notebook line
press Next
```

Not like this:

```text
scan a dashboard
interpret metrics
choose between many panels
```
