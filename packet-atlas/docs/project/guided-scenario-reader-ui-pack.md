# Packet Atlas v12.8–v13.2 — Guided Scenario Reader UI Pack

This patch turns the scenario selector into a scenario reader.

## v12.8 — One-step scenario reader

The learner reads one selected scenario step at a time.

## v12.9 — Evidence checklist

Each step shows only the evidence needed for that scenario step.

## v13.0 — Do-not-jump guard

Each step reminds the learner what not to debug too early.

## v13.1 — Notebook line

Each step gives a short line to remember.

## v13.2 — Next/Previous controls

Scenario reading becomes manual-first.

## Product rule

The selector chooses the story path. The reader guides one step at a time.

```text
choose scenario -> read step -> ask question -> check evidence -> write notebook line -> next
```
