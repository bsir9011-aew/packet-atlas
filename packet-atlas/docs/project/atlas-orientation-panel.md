# Packet Atlas v7.2 — Atlas Orientation Panel

v7.2 adds an orientation panel because Packet Atlas has become a layered system.

The panel answers:

- Where am I?
- Why did we use the terminal?
- What is the difference between the app and the lab work?
- Which workspace should I open first?
- What is each workspace for?

The product signal was clear: if the builder gets lost, a future learner will also get lost. The orientation panel makes the atlas explain itself before the user dives into the details.
