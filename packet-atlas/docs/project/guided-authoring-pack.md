# Packet Atlas v10.3–v10.7 — Guided Authoring Pack

This pack makes the guided story maintainable without adding UI dashboard weight.

## v10.3 — Full guided script model

Builds a full guided script from the current journey scenario.

## v10.4 — Markdown renderer

Renders the story into a readable script.

## v10.5 — Story quality rules

Checks that the story has evidence questions, do-not-jump guards and vocabulary helpers.

## v10.6 — Guided story reports

Adds command-line reports:

```bash
npm run story:guided
npm run story:quality
```

## v10.7 — Authoring contract

Documents the rule:

```text
story first, panels second
```

Focus Mode should guide. Atlas Mode can inspect.
