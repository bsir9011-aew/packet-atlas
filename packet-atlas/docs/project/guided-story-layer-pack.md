# Packet Atlas v9.8–v10.2 — Guided Story Layer Pack

This is a larger guided-experience pack.

It avoids dashboards and adds a stronger story layer.

## v9.8 — Story script

Each stage gets one spoken line.

## v9.9 — Mental model

Each stage explains how to imagine the concept.

## v10.0 — Evidence question

Each stage asks what proof would confirm the step.

## v10.1 — Do-not-jump guard

Each stage warns what not to debug too early.

## v10.2 — Vocabulary helper

Each stage gets one or two simple terms with "do not confuse with" guidance.

## Product rule

The user should read the screen like a guided technical story:

```text
spoken line
mental model
evidence question
do-not-jump warning
vocabulary
next step
```

This is not a metric dashboard.
