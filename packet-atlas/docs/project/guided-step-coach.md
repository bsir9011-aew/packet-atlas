# Packet Atlas v9.2 — Guided Step Coach

v9.2 strengthens the guided reading experience.

The user should not feel like they are operating a dashboard. They should feel guided through one journey.

## Reading pattern

```text
Before -> Now -> Next
Say it simply
What to do now
Proof question
Notebook line
```

## Product rule

The coach is not a quiz and not a course module.

It is a narrative layer that tells the user how to read the current stage before pressing Next.

## Why this matters

The atlas has many panels. The coach gives the user one job:

```text
understand this step first
```
