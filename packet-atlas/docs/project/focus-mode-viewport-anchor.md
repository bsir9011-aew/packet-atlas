# Packet Atlas v8.4 — Focus Mode Viewport Anchor

v8.4 polishes Focus Mode so the current animated step behaves like the anchor of the screen.

## What changed

- Focus Mode gives the Animated Journey panel a clearer current-step marker.
- Playback controls become sticky near the bottom while reading.
- The stage rail is centered.
- The workspace shell is constrained so the guided story does not feel scattered on wide screens.

## Why this matters

The app already has depth. In Focus Mode the user should feel:

```text
This is the step I am supposed to read now.
The rest of the atlas supports it.
```
