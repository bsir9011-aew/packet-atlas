# Packet Atlas v13.3–v13.7 — Full Stage Play Mode Pack

This patch changes the direction of Focus Mode.

## Product decision

Focus Mode is now treated as **Play Mode**.

The whole screen should refer to the current journey stage.

## Why

The user should not feel that they are scanning a dashboard.

They should feel:

```text
I am playing/reading one current stage.
The title tells me what this stage is.
Everything on screen supports this stage.
I press Next when I understand it.
```

## v13.3 — Play Mode framing

Adds a strong Play Mode banner in Focus Mode.

## v13.4 — Current-stage screen

The current stage title becomes the dominant screen anchor.

## v13.5 — Hide dashboard leftovers

Scenario selector, rails, maps, inspectors and detail panels are hidden in Play Mode.

## v13.6 — Sticky play controls

The bottom controls become the manual step action area.

## v13.7 — Product contract

Atlas Mode can inspect. Play Mode teaches one stage.

```text
Atlas Mode = browse and inspect
Play Mode = read current stage and press Next
```
