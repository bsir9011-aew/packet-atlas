# Packet Atlas v18.0 — Play Mode Layer Ladder

v18.0 adds a layer ladder to Play Mode.

## Goal

The user should repeatedly map the same journey across layers:

- human intent,
- application meaning,
- TLS protection,
- transport identity,
- network routing,
- local link delivery,
- physical signals.

## UX rule

The current stage highlights active lenses from `stage.layerFocus`. Inactive layers remain visible as background context.
