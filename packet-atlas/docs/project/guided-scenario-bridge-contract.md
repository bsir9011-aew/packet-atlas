# Guided Scenario Bridge Contract

A scenario bridge anchor must answer:

1. Which scenario does this step belong to?
2. Which journey stage should the learner look at?
3. Why does this step map to that stage?
4. What should the learner read or prove there?

This keeps future UI simple:

```text
scenario selection does not create a separate app;
it selects a guided path through the same journey.
```
