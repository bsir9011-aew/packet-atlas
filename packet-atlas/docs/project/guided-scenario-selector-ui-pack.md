# Packet Atlas v12.3–v12.7 — Guided Scenario Selector UI Pack

This patch adds the first UI bridge for guided scenarios.

## v12.3 — Scenario selector

Adds a simple scenario selector to Animated Journey Mode.

## v12.4 — Symptom-first cards

Each scenario starts from what the user sees.

## v12.5 — First diagnostic question

The selected scenario shows the first question the learner should ask.

## v12.6 — Runtime reading steps

The selected scenario exposes guided runtime steps.

## v12.7 — Focus-mode restraint

In Focus Mode, the selector stays compact and hides detailed step lists.

## Product rule

The selector is not a dashboard.

It answers:

```text
Which story path am I following?
What symptom starts it?
What is the first question?
```
