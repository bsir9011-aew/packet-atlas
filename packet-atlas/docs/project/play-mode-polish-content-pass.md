# Packet Atlas v16.1 — Play Mode Polish Content Pass

v16.1 expands the Polish content dictionary and applies the text display mode more broadly inside Play Mode.

## What changed

- Stage labels now have Polish equivalents.
- Common Play Mode narrator, coach and study snapshot text has Polish equivalents.
- `textDisplayMode` is now used across more Play Mode content, not only the hero title.
- Technical names such as HTTP GET, TLS, TCP, Router/NAT can remain technical where that is clearer.

## Product rule

The dictionary remains a safe transitional layer. Missing translations still fall back to English instead of rendering empty text.
