# Packet Atlas v8.2 — Presentation Focus Mode Polish

v8.2 adds a Focus Mode for Animated Journey Mode.

The goal is not to remove atlas depth. The goal is to make the current step readable when the user wants a guided story.

## What Focus Mode does

- keeps Animated Journey Mode as the primary reading path,
- hides top-level guidance panels while focus is active,
- hides workspace tabs and early filter controls,
- enlarges the active step area,
- keeps supporting map/inspector panels below the guided story,
- adds a sticky focus ribbon so the user knows why the screen is simpler.

## Product rule

Focus Mode is not a course mode, not a quiz and not a separate route.

It is a calmer view of the same atlas.

## Why this follows v8.1

After branch choices and the DNS failure path, the screen has enough value to become dense again. Focus Mode gives the user a way to say:

```text
Do not show me everything.
Guide me through the journey.
```
