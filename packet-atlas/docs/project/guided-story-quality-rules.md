# Guided Story Quality Rules

A guided step is good when it has:

- one spoken line,
- one mental model,
- one evidence question,
- one do-not-jump guard,
- one short notebook line,
- one handoff to the next step.

A guided step is bad when it becomes:

- a metric dashboard,
- a quiz module,
- a disconnected lecture,
- a list of unrelated panels.

Packet Atlas should preserve:

```text
one journey, many lenses
```
